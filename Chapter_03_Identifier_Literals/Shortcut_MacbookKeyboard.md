# Macbook Keyboard Shortcuts for VS Code

> **NOTE:** Replace `Cmd` with `Ctrl` if you're on Windows/Linux

---

## 1. General Shortcuts

| Action | Keyboard Shortcut |
|--------|-------------------|
| Command Palette | `Cmd + Shift + P` |
| New File | `Cmd + N` |
| Open File | `Cmd + O` |
| Save File | `Cmd + S` |
| Save As | `Cmd + Shift + S` |
| Close File | `Cmd + W` |
| Close Folder | `Cmd + K Cmd + W` |
| Open New Window | `Cmd + Shift + N` |
| Split Editor | `Cmd + \` (backslash) |
| Toggle Sidebar | `Cmd + B` |
| Toggle Terminal | `Ctrl + `` (backtick) |
| Open Settings | `Cmd + ,` |
| Open Keyboard Shortcuts | `Cmd + K Cmd + S` |

## 2. Editing Shortcuts

### Cut, Copy, Paste
| Action | Keyboard Shortcut |
|--------|-------------------|
| Cut | `Cmd + X` |
| Copy | `Cmd + C` |
| Paste | `Cmd + V` |

### Undo/Redo
| Action | Keyboard Shortcut |
|--------|-------------------|
| Undo | `Cmd + Z` |
| Redo | `Cmd + Shift + Z` or `Cmd + Y` |

### Selection
| Action | Keyboard Shortcut |
|--------|-------------------|
| Select All | `Cmd + A` |
| Select Line | `Cmd + L` |
| Select Word | `Cmd + D` (select next occurrence) |
| Select All Occurrences | `Cmd + Shift + L` |

### Duplicate & Delete
| Action | Keyboard Shortcut |
|--------|-------------------|
| Duplicate Line | `Opt + Shift + Down` (or Up) |
| Delete Line | `Cmd + Shift + K` |
| Delete Word | `Opt + Delete` (backward) or `Opt + Fn + Delete` (forward) |

### Indentation & Comments
| Action | Keyboard Shortcut |
|--------|-------------------|
| Increase Indent | `Cmd + ]` |
| Decrease Indent | `Cmd + [` |
| Toggle Line Comment | `Cmd + /` |
| Toggle Block Comment | `Opt + Shift + A` |

### Move Lines
| Action | Keyboard Shortcut |
|--------|-------------------|
| Move Line Up | `Opt + Up` |
| Move Line Down | `Opt + Down` |
