# Windows Keyboard Shortcuts for VS Code

> **NOTE:** These shortcuts are for Windows. For macOS, use `Cmd` instead of `Ctrl` and `Alt` instead of `Shift+Alt`

---

## 1. General Shortcuts

| Action | Keyboard Shortcut |
|--------|-------------------|
| Command Palette | `Ctrl + Shift + P` |
| New File | `Ctrl + N` |
| Open File | `Ctrl + O` |
| Save File | `Ctrl + S` |
| Save As | `Ctrl + Shift + S` |
| Close File | `Ctrl + W` |
| Close Folder | `Ctrl + K Ctrl + W` |
| Open New Window | `Ctrl + Shift + N` |
| Split Editor | `Ctrl + \` (backslash) |
| Toggle Sidebar | `Ctrl + B` |
| Toggle Terminal | `` Ctrl + ` `` (backtick) |
| Open Settings | `Ctrl + ,` |
| Open Keyboard Shortcuts | `Ctrl + K Ctrl + S` |

## Pro Tips

✅ **Memorize the Top 5:** These will boost productivity the most
- `Ctrl + Shift + P` - Command Palette
- `Ctrl + P` - Quick Open
- `Ctrl + F` - Find
- `Ctrl + /` - Toggle Comment
- `Ctrl + S` - Save

✅ **Use Command Palette:** If you forget a shortcut, use `Ctrl + Shift + P` and search for the command

✅ **Customize Shortcuts:** Go to `Ctrl + K Ctrl + S` to view and customize keyboard shortcuts

✅ **Learn Incrementally:** Don't try to memorize all shortcuts at once. Pick a few and practice daily

## Key Differences from macOS

| Action | macOS | Windows |
|--------|-------|---------|
| Primary Modifier | `Cmd` | `Ctrl` |
| Secondary Modifier | `Opt` | `Alt` |
| Alternative Secondary | `Shift + Opt` | `Shift + Alt` |
| Redo | `Cmd + Shift + Z` | `Ctrl + Shift + Z` or `Ctrl + Y` |
| Toggle Terminal | `` Ctrl + ` `` | `` Ctrl + ` `` (same) |
| Format Document | `Opt + Shift + F` | `Shift + Alt + F` |
| Quick Fix | `Cmd + .` | `Ctrl + .` |
