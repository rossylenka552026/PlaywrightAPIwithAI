## JavaScript Naming Conventions Covered

| Convention | Pattern | Use Case |
|------------|---------|----------|
| camelCase | `firstName` | Variables & functions |
| PascalCase | `UserProfile` | Classes & constructors |
| snake_case | `first_name` | Alternative style |
| SCREAMING_SNAKE_CASE | `MAX_USERS` | Constants |
| kebab-case | `user-name` | CSS, HTML, URLs (NOT JS variables) |
| Hungarian Notation | `strName` | Outdated — avoid |

## Comment Types Covered

| Type | Syntax | Shortcut (macOS) | Shortcut (Windows) |
|------|--------|-------------------|--------------------|
| Single-line | `// comment` | `Cmd + /` | `Ctrl + /` |
| Multi-line | `/* comment */` | `Opt + Shift + A` | `Shift + Alt + A` |
| JSDoc | `/** comment */` | `Opt + Shift + A` | `Shift + Alt + A` |
| Special markers | `// TODO:` | `Cmd + /` | `Ctrl + /` |

## Getting Started

1. **Clone the repository:**
   ```bash
   git clone https://github.com/rossylenka552026/PlaywrightAPIwithAI.git
   cd PlaywrightAPIwithAI
   ```

2. **Install dependencies (when applicable):**
   ```bash
   npm install
   ```

3. **Open in VS Code:**
   ```bash
   code .
   ```

4. **Run a JavaScript file:**
   ```bash
   node Chapter_01_Basics/01_Basics.js
   ```
